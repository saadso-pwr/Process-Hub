# Process Hub

Next.js 16 app (App Router) with TypeScript, Tailwind CSS v4, and ESLint.

## Stack
- **Framework**: Next.js 16 (App Router) — see `node_modules/next/dist/docs/` for current API docs
- **Styling**: Tailwind CSS v4 via `@tailwindcss/postcss`
- **Language**: TypeScript strict mode

## Figma MCP
Figma MCP is configured in `.mcp.json`. Set `FIGMA_API_KEY` env var (Figma Personal Access Token) before using Figma tools.

Generate a token at: Figma → Account Settings → Personal Access Tokens

## Commands
- `npm run dev` — start dev server (localhost:3000)
- `npm run build` — production build
- `npm run lint` — run ESLint
